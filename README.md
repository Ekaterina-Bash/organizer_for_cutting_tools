# Органайзер для канцелярских ножей, скальпелей, ножниц и лезвий.
Для поддержания порядка в рабочей зоне давайте создадим простой и удобный в использовании органайзер. В конкретном случае, это органайзер для режущих предметов.

# **:wrench: Создание настенного органайзера для хранения режущих предметов**

## **:mag: Обзор решения**
Универсальным и наиболее удобным решением для хранения большого количества инструментов является использование стенки типа skadis. Такая конструкция обеспечивает эргономичность, экономию пространства и удобство в использовании. Итак, наше изделие может быть выполнено при помощи разных технологий производства. Кроме того, оно должно встраиваться в концепцию данной стены и соответствовать главным критериям:
- мобильность;
- компактность;
- удобство использования;
- прочность и долговечность;
- экономность в использовании материала.

Режущие предметы включают в себя: канцелярские ножи, скальпели, ножницы и лезвия.

## **:hammer:Дизайн модели и производство**

### 1. Описание модели, выбор технологии производства.

Ножи наиболее удобно хранить в вертикальном положении, так их значительно удобнее доставать. Поэтому смоделируем органайзер в виде некой коробки, разделенной на отсеки, специально для разных типов режущих. Чтобы были видны инструменты, переднюю стенку выполним из прозрачного акрила. Поскольку важна безопасность при взаимодействии с режущими предметами, их стоит острой частью опускать вниз. Чтобы контейнер не царапался и не изнашивался, проложим вспененный материал на дно органайзера. Ножницы намного больше остальных режущих, поэтому их лучше расположить отдельно, на крючке.

Крепления и крючок для ножниц произведем путем 3D печати, остальные детали получим резкой лазером из фанеры и акрила толщины 3мм.

### 2. Проектирование модели.

Произведя измерения параметров всех режущих предметов, прикинем размеры органайзера. Затем, используя программу Rhino, создаем всю модель и эскизы каждой из деталей.

### 3. Процесс создания

- крепления

Крепления напечатаны пластиком PETG.
При подготовке детали к печати, а именно ее расположении на рабочем столике, необходимо учитывать нагрузку на крепления в соотвествии с расположением слоёв пластика.

Для получения подставки необходим её чертеж в формате .dxf. Далее, нужно отредактировать чертеж в CorelDraw и задать нужные настройки для лазерной резки по фанере толщиной 3мм и 6мм.

### 4. Сборка

Детали резки необходимо укрепить между собой. Это можно достичь использованием клея ПВА первым этапом, а вторым этапом - клея-пистолета. Акриловую стенку тоже закрепить на клей-пистолет, поскольку он прозрачный и не оставит разводов на передней стенке.

## **:key: Как использовать**

Для создания этого органайзера следуйте шагам:

1. Скачайте модель крепления, переведите в формат .stp или .stl, используя слайсер, отравьте на печать
2. Скачайте модель крепления для оловоотсоса, используя слайсер, отправьте на печать
3. Скачайте файл верхней подложки держателя и осуществите лазерную резку (расчёт на фанеру 6 мм)
4. Скачайте файл нижней подложки держателя и осуществите лазерную резку (расчёт на фанеру 3 мм)
5. Совершите сборку деталей с помощью винтов м3 и болта м4х70
